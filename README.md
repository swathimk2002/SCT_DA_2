 # Data Cleaning – Global Superstore Dataset

##  Overview
This project focuses on cleaning and preparing the Global Superstore dataset using Python and Pandas to make it ready for analysis.

---

##  Tools Used
- Python  
- Pandas  
- Google Colab  

---

##  Steps Performed
- Extracted dataset from ZIP file  
- Loaded data using Pandas  
- Inspected data using `df.head()` and `df.info()`  
- Identified missing values  
- Dropped **Postal Code** column (only column with missing values)  
- Removed duplicate rows  
- Cleaned column names  
- Converted date columns to datetime format  

---

## Files
- data_cleaning (2).ipynb
- Cleaned_Globalsuperstore (1).csv

---

## Outcome
 Cleaned dataset for analysis

---

##  What I Learned
- Importance of cleaning data before analysis  
- How to identify and handle missing values  
- Removing duplicate data to improve accuracy  
- Using Pandas for data preprocessing  
- Converting data types for better analysis

##  Conclusion
The dataset was cleaned and prepared by removing unnecessary data, fixing structure, and ensuring it is ready for further analysis.
